# Keiba App

## 概要

競馬の出馬表を取得し、お気に入りの馬や騎手が出走する際にDiscordへ通知を送るWebアプリです。

Spring Bootの学習およびポートフォリオ作成を目的として開発しています。

---

## 主な機能

### レース情報取得

- Yahoo!競馬から出馬表情報を取得
- レース一覧表示
- 開催がない日はダミーデータを表示

### お気に入り馬管理

- お気に入り馬の登録
- お気に入り馬一覧表示
- お気に入り馬削除
- 重複登録防止

### お気に入り騎手管理

- お気に入り騎手の登録
- お気に入り騎手一覧表示
- お気に入り騎手削除
- 重複登録防止

### Discord通知

- お気に入り馬の出走通知
- お気に入り騎手の騎乗通知
- Discord Webhook連携

### 通知管理

- 通知履歴保存
- 重複通知防止

### 定期実行

- Spring Schedulerによる自動実行
- 指定時刻に出走チェック

## 使用技術

| 技術 | 内容 |
|--------|--------|
| Java | 21 |
| Spring Boot | 3.3 |
| Spring Data JPA | ORM |
| Thymeleaf | テンプレートエンジン |
| H2 Database | 開発用DB |
| Jsoup | スクレイピング |
| Maven | ビルド管理 |
| Discord Webhook | 通知機能 |
| Git / GitHub | バージョン管理 |

---

## システム構成

```text
Yahoo!競馬
    ↓
 Jsoup
    ↓
RaceService
    ↓
Thymeleaf画面

お気に入り馬・騎手
    ↓
H2 Database
    ↓
RaceNotificationService
    ↓
Discord通知
```
